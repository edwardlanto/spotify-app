# Spotify Clone

Spotfify Clone using Javascript (NextJS, TailwindCSS, Recoil, Spotify Web API). 
[Demo](https://spotify.edwardl.ca/)

## Core packages

1. Recoil - State Management
2. Next Auth - Authentication
3. Tailwind CSS - Styled Components
4. Hero Icons - Icons
